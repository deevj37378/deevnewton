# deevnewton
This repo contains a static frontend clone of the Newton School of Technology website created with semantic HTML5 and CSS (Flexbox + Grid). The goal is to practice layout, responsive design, and UI implementation without using JS frameworks. It reproduces visual styles, layout, and responsiveness for learning purposes.
Pixel-inspired clone of public Newton School pages (landing/home, hero, features, course cards, testimonials, footer)
<b>Features</b>
Responsive: mobile / tablet / desktop breakpoints

Semantic HTML5 markup

CSS organized by components (header, hero, cards, footer)

Image placeholders and assets included

No JavaScript required for the core UI (pure HTML/CSS)
